e-PlantShopping 🌿

A modern and responsive plant shopping application inspired by Paradise Nursery, built with React. The application allows users to browse different categories of houseplants, add plants to a shopping cart, adjust quantities, and view the total cost before checkout.

Features

🌱 Landing Page

- Beautiful background image
- Company name and introduction
- Information about Paradise Nursery
- Get Started button to navigate to the product listing page

🪴 Product Listing Page

- Plants organized into multiple categories
- Thumbnail image for each plant
- Plant name and price
- Add to Cart functionality
- Dynamic shopping cart icon displaying the total number of items
- Navigation between pages

🛒 Shopping Cart Page

- Displays all items added to the cart
- Shows plant image, name, and unit price
- Displays quantity and total cost for each item
- Increase and decrease quantity buttons
- Remove item from cart
- Total number of plants
- Total cart value
- Continue Shopping button
- Checkout button

Installation

Clone the repository:

git clone <repository-url>

Navigate to the project directory:

cd e-PlantShopping

Install dependencies:

npm install

Run the development server:

npm run dev

Open the local URL displayed in the terminal.
